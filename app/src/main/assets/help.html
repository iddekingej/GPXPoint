GPS Location is a simple app that displays the following information
  • Position (latitude/longitude)
  • Speed
  • Altitude
  • Accuracy
  • Number of satellites

The GPS information is displayed after turning on "Display GPS position". Disabling it will save battery life.

Sometimes temporarily the following message is displayed:"Warning: GPS is temporarily unavailable". This is normal behaviour

On the right of the screen near the "Position" and "Other" section title, there is a small icon. Clicking oh this icon will copy the information to the clipboard.

This program doesn't need any network/internet connection

Privacy statement
This program doesn't collect any information and doesn't sends any information to a third party.
This program uses the android Location service. It's possible due to phone configuration that this service sends some anonymous information to google.
