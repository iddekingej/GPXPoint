Usage
"GPS Location" is a simple app that displays the following information:
  • Position (latitude/longitude)
  • Speed
  • Altitude
  • Accuracy
  • Number of satellites

The GPS information is displayed after turning on the option "Display GPS position". Turning of this option will save battery life.

Sometimes the following message is displayed temporarily: "Warning: GPS is temporarily unavailable". This is normal behavior.

The information displayed in the app can easily be easily copied to the clipboard by clicking on the following icon:


This program doesn't need any network/internet connection
